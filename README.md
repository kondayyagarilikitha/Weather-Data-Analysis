# 🌦️ India Weather Data Analysis & Prediction System

## 📌 Project Overview

This project is a **Weather Data Analysis and Prediction System** built using Python and Streamlit.  
It analyzes historical weather data from Indian states and districts and predicts future temperature trends using Machine Learning.

The system provides an interactive web dashboard for visualization, filtering, and forecasting.

---

## 🎯 Objectives

- Analyze historical weather data 📊  
- Visualize temperature trends over time 📈  
- Build a Machine Learning model for prediction 🤖  
- Forecast future temperature values 🔮  
- Create an interactive Streamlit dashboard 🌐  

---

## 📂 Dataset Information

The dataset contains India-based weather records with the following columns:

- State/UT  
- District  
- Temperature (°C)  
- Humidity (%)  
- Wind Speed (km/h)  
- Condition  
- Last Updated

## 🚀 Features

### 📊 Data Analysis
- State-wise and district-wise filtering

### 📈 Visualization
- Temperature trend graphs

### 🤖 Machine Learning
- Linear Regression / Random Forest model
